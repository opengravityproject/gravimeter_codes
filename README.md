# Mag-Grav-2 Gravimeter

Mag-Grav-2 is a magnetic gravimeter design built around a suspended reflector, magnetic levitation geometry, a laser path, and camera-based centroid capture. This iteration is intended for stable, repeatable, longer-duration operation.

## Repository Contents

This repository contains early gravimeter support code and documentation.

- `interval_capture_2.py` - Captures time-spaced images from a camera for gravimeter centroid recording.
- `gravimeter_instruction.md` - General instructions for preparing and running the gravimeter capture workflow.
- `LICENSE` - The project license.

The code is focused on image capture for experimental gravimeter measurements.
